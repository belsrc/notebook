## Web Feature Service (WFS) Formats

Web Feature Service (WFS) offers direct fine-grained access to viewport based geographic information at the feature and feature property level. A WFS describes discovery, query, or data transformation operations and is a Open Geospatial Consortium standard. Returned values can be GML, GeoJSON, Shapefile, CSV, JSON, etc.

### Formats

- GML2
- GML3
- GeoJSON: GeoJSON format
- ESRIGEOJSON: Esri GeoJSON format
- Shapefile
- CSV
- JSON
- JSONP

## Vector Tile Service (VTS)

Vector Tile Services (VTS) enables you to share and consume vector tiles. Vector tiles are packets of geographic data, packaged into pre-defined roughly-square shaped "tiles" for transfer over the web.

### Formats

- GeoJSON
- TopoJSON
- MapBox Vector

## Web Map Service (WMS) Formats

A Web Map Service (WMS) is a standard protocol developed by the Open Geospatial Consortium in 1999 for serving geo-referenced map images over the Internet. A WMS request defines the geographic layer(s) and area of interest to be processed. The response to the request is one or more geo-registered map images (JPEG, PNG, etc).

### Formats

- Bitmap (PNG, JPEG, etc)
